WELCOME TO VIMTUTOR

Lesson 1.1: MOVING THE CURSOR - Use h, j, k, l to move left, down, up,
right.

Lesson 1.2: EXITING VIM - Use :q to quit. - Use :wq to write and quit.

Lesson 1.3: TEXT EDITING - DELETION - Use x to delete a character.

Lesson 1.4: INSERTING TEXT - Use i to insert before the cursor. - Use a
to insert after the cursor. - Use o to open a new line.

Lesson 1.5: EDITING A FILE - Use :w to save the file. - Use :!command to
run shell commands.

Lesson 1.6: GETTING HELP - Use :help to open help.
